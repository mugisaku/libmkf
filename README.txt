これは構文解析ライブラリです
読み込んだ定義ファイルに従って、テキストを解析し、構文木を生成します


test_mkfはmkffmt.def.txtを定義ファイルとして読み込み、
続いてmkffmt.def.txt自体をテキストとして解析して、出力します
mkffmt.def.txtは定義ファイルの書式を定義ファイル化したものです


randjsonはjson書式のランダムなデータを
sample.jsonというファイル名で生成します


print_mkfは定義ファイルとテキストファイルを受け取り、
解析して出力します


test_mkfjsonはjson.def.txtを定義ファイルとして読み込み、
json書式のランダムなデータを構文木にするテストをします
json.def.txtはjson書式を定義ファイル化したものです
